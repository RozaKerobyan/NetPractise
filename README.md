# NetPractise

## 📌 Description

NetPractice is a project designed to teach the fundamentals of networking through hands-on practice.
It consists of multiple levels where you must configure network settings correctly to ensure communication between devices.

The main goals of this project are:

- Understand TCP/IP addressing
- Learn how subnet masks work
- Understand the role of a default gateway
- Work with routers and switches
- Apply OSI model concepts in real scenarios

## ⚙️ Instructions
### How to Run ▶️
1. Open the project directory
2. Run the following command in the terminal: ```run.sh ```
3. Then open the training or evaluation interface.

### 📁 Submission Requirements
After completing each level, you must export your configuration
- Use the export function provided in the NetPractice interface
- Save the file in the required format (usually .json)
- You must provide 10 configuration files in the root of your repository
- Example:

```bash
level1.json
level2.json
...
level10.json
```

## 🧠 Resources
### 📚 Networking Concepts Covered

This project includes the following key networking topics:

- TCP/IP addressing
- Subnet masks
- Default gateway
- Routers and switches
- OSI layers (Layer 1 - Physical to Layer 7 - Application)
- IP routing and packet forwarding
